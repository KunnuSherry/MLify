## ✨ **Features**

<div align="center">

| Feature | Description
|---------|-------------
| 📤 **Dataset Upload** | Upload CSV/Excel files and select target variable | ✅ Active |
| 📊 **Automatic Analysis** | EDA, correlation heatmaps, missing values, feature importance | ✅ Active |
| 🤖 **Model Training** | Train 5 models (Linear/Logistic, Decision Tree, Random Forest, XGBoost, SVM) | ✅ Active |
| 🎯 **Auto Preprocessing** | Handle missing values, encoding, scaling automatically | ✅ Active |
| 💾 **Model Access** | Download trained model or generate API key | ✅ Active |
| 📄 **Reports** | Download analysis and prediction reports (PDF/HTML) | ✅ Active |

</div>
